FizzBuzz!

View the code

FizzBuzz prints out 'Fizz' if a number is divisible by 3 and 'Buzz' if a number is divisible by 5. If a number is divisible by both 5 and 3 'FizzBuzz' is printed!

Instructions:

Enter two numbers between 1 and 100. The first must be lower than the second and they must both be integers.
